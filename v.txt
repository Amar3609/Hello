Practical 1 — HTML Text Formatting Tags

Viva Questions

Q1. What is HTML?

HTML stands for HyperText Markup Language.

It is used to create web pages.


Q2. What is a tag in HTML?

Tags are special keywords enclosed in < >.

Example: <h1>, <p>.


Q3. Difference between <b> and <strong>?

<b> makes text bold.

<strong> gives importance to text.


Q4. Use of <i> tag?

Makes text italic.


Q5. Use of <u> tag?

Underlines text.


Q6. What is the use of <sub>?

Displays subscript text.

Example: H₂O.


Q7. What is the use of <sup>?

Displays superscript text.

Example: x².


Q8. What is <hr> tag?

Creates horizontal line.


Q9. What is <br> tag?

Moves content to next line.


Q10. What is the purpose of <title> tag?

Sets webpage title on browser tab.
